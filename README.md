# Klosterke

## Installing and running the original assignment
`cd ./casus`

`composer install`

`./vendor/bin/kahlan`

## Installing and running the completed assignment
`cd ./assignment`

`composer install`

`./vendor/bin/kahlan`

## Installing and running the Laravel+Vue App
`cd ./klosterke-laravel`

`composer install`

`npm install`

`npm run dev`
